Visualisation de plusieurs dépôts git

Oct 04, 2022 - git

Bonjour,

pour ce premier post, un petit utilitaire pour faciliter la vie. J’utilise dans la plupart de mes projets des sauvegardes par git. J’ai, en effet, un mini serveur personnel qui me permet d’ajouter un point de sauvegarde. Je ne rentrerais pas dans le débat des serveurs externes non personnels.

Bref, j’ai quand même un petit nombre de dépôts sur mon pc et aller voir dans chaque dépôt son état serait un peu long. Actuellement, j’en ai 5 en cours mais je compte encore étendre ce nombre.

Dans un premier temps, je souhaite simplement voir leurs états. Comme pré-requis, il suffit que tous les dépôts soient dans un même dossier. Il suffit de se mettre dans ce dossier et taper la commande :

mgitstatus

pour avoir toutes les informations

Bien sûr avant, il est nécessaire de l’installer. Les informations, mêmes écrites en anglais, sont compréhensibles. L’utilitaire est disponible sur github :

Multi-git-status

Je ne sais pas si un utilitaire pour tout faire serait bien. Ce sera à voir avec l’utilisation future et si le nombre augmente.
